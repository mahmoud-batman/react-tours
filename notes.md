# Deployment on `Netlify` :

## 1 ) Drag and Drop :

1- Go to netlify.com

2- npm run build

3- Drag and Drop the build folder

## 2 ) Continous Deployment with `Git` :

**1- Git :**

- Initialize the local repo.

  > git init

- Add files to Staging Area

  > git add .

- Commit Changes

  > git commit -m

  - \- m : message

  > add remote url

- Push changes to remote repo
  > git push -u origin master
